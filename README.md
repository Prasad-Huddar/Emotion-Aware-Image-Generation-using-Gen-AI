# Emotion Aware Image Generation using Gen AI

This repository contains notebooks for emotion-aware image generation using generative AI techniques.

## Contents

- emotion-image-generation.ipynb
- emotion-image-pipeline.ipynb

## Purpose

These notebooks demonstrate an approach for generating images based on emotional context and related generative AI workflows.

## Getting Started

1. Open the notebooks in Jupyter Notebook or JupyterLab.
2. Install the required dependencies listed in the notebook environment.
3. Run the cells in order to explore the pipeline.

## Notes

This project is intended for experimentation and educational use.
